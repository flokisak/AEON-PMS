## ğŸ› ï¸ Installation & Setup

### Prerequisites
- Node.js 18+ 
- PostgreSQL 14+
- Docker (optional)

### Quick Start
```bash
# Clone the repository
git clone https://github.com/your-org/hotel-management-system.git
cd hotel-management-system/pms-core

# Install dependencies
npm install
```
